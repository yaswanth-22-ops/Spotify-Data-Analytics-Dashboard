📌 Project Overview

This project is an interactive Spotify Data Analytics Dashboard built using Power BI.
It analyzes music streaming data to uncover trends in song popularity, artist performance, and user listening behavior.

The dashboard helps transform raw Spotify data into meaningful insights through visual storytelling.

📂 Dataset Description

The dataset includes the following features:

Track Name
Artist Name
Album
Release Year
Popularity Score
Genre
Audio Features:
Danceability
Energy
Tempo
Loudness
Valence
📊 Key Insights
🎯 Top artists contribute the majority of total streams
📅 Certain years show spikes in hit song releases
🎵 High energy & danceability correlate with popularity
🌍 Genre preferences vary significantly over time
